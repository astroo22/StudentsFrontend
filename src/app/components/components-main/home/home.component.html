My name is Jonathan Starkey and I am a Full Stack Engineer with three years of experience. I have created this website as a general showcase of things a company would commonly want in a website dashboard. As a result this website is also a basic full stack application so I hope you will take your time and look around.
Technologies & Tools Used
Frontend:
  • Angular
  • Hosted on AWS S3 Bucket
  • Integrated with CloudFront
Backend:
  • Golang
  • Hosted on AWS EC2
  • Integrated with CloudFront
Database:
  • PostgreSQL
  • Hosted on AWS RDS
API:
  • RESTful Architecture
Middleware & Security:
  • Gorilla Mux for Routing
  • Custom middleware for CORS management
  • JWT (JSON Web Token) for authentication and authorization
  • HTTPS using AWS route 53 for certification
Continuous integration & Development:
  • Frontend & Backend CI/CD pipes using AWS CodePipeline, CodeBuild and CodeDeploy
